# Linux环境检测脚本使用介绍

第八次修订：2019/10/08

修订人：Kevin Lee 

工具版本：v1.0.1 

脚本语言：bash shell



此脚本适用Linux环境：

```
- CentOS 6.x~7.x
- Ubuntu 14.x~18.x
```

此脚本运行命令如下：

```
bash sys_env_check.sh
```

此脚本可检测Linux类型：

```
- CPU信息
- 内存信息
- IP信息
- 路由信息
- 磁盘信息
- 系统信息
- 服务启动配置
- 服务运行情况
- 端口监听
- 僵尸进程
- Docker基础信息
- Docker运行情况
- 系统登陆用户
- hosts信息
- CPU占用top10
- 内存占用top10
....
```
