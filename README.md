# Linux环境检测脚本使用介绍

第二次修订：2019/09/30

修订人：Kevin Lee 

工具版本：v1.0.0 

脚本语言：bash shell



此脚本适用Linux环境：

```
- CentOS 6.x~7.x
- Ubuntu 14.x~18.x
```

此脚本运行命令如下：

```
bash sys_env_check.sh
```

此脚本可检测Linux类型：

```
- CPU信息
- 内存信息
- IP信息
- 路由信息
- 磁盘信息
- 系统信息
- Docker运行信息
- 服务运行情况
- 僵尸进程
- 端口监听
- CPU占用top10
- 内存占用top10
....
```
